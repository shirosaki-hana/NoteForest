# Utils

이 폴더에는 유틸리티 함수들을 저장합니다.

## 사용법

재사용 가능한 헬퍼 함수들을 적절히 분류하여 저장하세요:

```
utils/
├── api.ts           # API 호출 관련 함수
├── format.ts        # 포맷팅 함수 (날짜, 숫자 등)
├── validation.ts    # 유효성 검증 함수
├── constants.ts     # 상수 정의
└── helpers.ts       # 기타 헬퍼 함수
```

모든 함수는 TypeScript로 작성하고 적절한 타입을 지정하세요.
